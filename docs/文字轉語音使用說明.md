# 文字轉語音使用說明

此功能目前使用 F5-TTS 做本機測試，可用 `voice-samples/` 裡的短音檔當參考聲音。

預計輸入：

- 文字稿
- 參考聲音音檔，若使用 voice cloning，放在 `voice-samples/`

預計輸出：

- `.wav` 或 `.mp3`
- 預設放在 `generated-audio/`

預計腳本：

```text
scripts\synthesize-voice-f5.ps1
```

## 產生測試語音

```powershell
.\scripts\synthesize-voice-f5.ps1 -Text "各位同學大家好，今天我們一起練習中文語音合成。"
```

## 新電腦準備

第一次在新電腦使用：

```powershell
.\scripts\setup-f5-tts.ps1
```

然後手動放入參考聲音檔：

```text
voice-samples\clean-ref-zh.wav
```

對應文字已提交在：

```text
voice-samples\clean-ref-zh.txt
```

可以這樣產生新音檔：

```powershell
$refText = Get-Content -Raw -LiteralPath ".\voice-samples\clean-ref-zh.txt"
.\scripts\synthesize-voice-f5.ps1 -RefAudio ".\voice-samples\clean-ref-zh.wav" -RefText $refText -Text "各位同學大家好，這是一段新的中文語音測試。"
```

F5-TTS 的中文流程比較偏簡體中文。若產生結果聽起來不像中文，先把測試文字改成簡體中文再試：

```powershell
.\scripts\synthesize-voice-f5.ps1 -Text "各位同学大家好，今天我们一起练习中文语音合成。"
```

預設會使用：

```text
voice-samples\20250420_____01_20s.wav
```

輸出會放在：

```text
generated-audio\
```

## 聲音範例建議

建議準備幾段短音檔：

- 每段 10 到 30 秒
- 背景安靜
- 只有你的聲音
- 不要有音樂、掌聲、多人對話
- 語速自然，不要刻意朗讀得太誇張

長錄音可以先放在 `audio-inbox/`，再切出適合的片段放到 `voice-samples/`。

## 從長錄音切片

如果已經有一段長錄音，可以用：

```powershell
.\scripts\split-voice-samples.ps1 ".\audio-inbox\20250420練講三寶.m4a"
```

預設會從第 60 秒、第 300 秒、第 600 秒各切出 20 秒，輸出到 `voice-samples/`。

也可以自己指定開始秒數和長度：

```powershell
.\scripts\split-voice-samples.ps1 ".\audio-inbox\20250420練講三寶.m4a" -StartSeconds 120,420,900 -DurationSeconds 15
```

如果剛安裝 FFmpeg 後系統還找不到 `ffmpeg`，可以先關掉並重開終端機，或用 `-FfmpegPath` 指定 `ffmpeg.exe`。
